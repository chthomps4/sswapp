# AI Workflow Services Business Model

Status: internal draft for Fred and CEO/Owner review
Owner route: CEO/Owner -> Fred -> Seth research -> Fred decision -> Ed/Jeff workflow -> Sara audit -> Alvin implementation if approved
Related Linear issues: SIG-20, SIG-26

## Idea

Signal Workshop can offer AI workflow setup for businesses using the same kind of operating system being built inside SSWApp: intake, task routing, review gates, content operations, research handoffs, audits, and implementation queues.

The internal business hub becomes a proof-of-work example, not a public promise yet.

## Service Concept

Help a business turn scattered requests, files, content ideas, support tasks, and implementation needs into a simple operating workflow using a practical tool stack:

- Slack or existing chat for intake.
- Linear or existing tracker for execution.
- Codex for review-ready drafts, docs, coding support, and summaries.
- GitHub for source control when code exists.
- Vercel for approved web app deployment when needed.
- Chrome/browser verification for UI and workflow checks.
- OpenAI platform tooling for approved AI features.
