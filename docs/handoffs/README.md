# Handoffs

Every meaningful task should produce a handoff. Use `HANDOFF_TEMPLATE.md`.

## Folders

- `inbox/`: untriaged handoffs.
- `fred/`: reports and decisions for Fred.
- `jeff/`: automation/workflow handoffs.
- `sara/`: audit and workspace efficiency handoffs.
- `seth/`: R&D handoffs.
- `ed/`: workflow designs for Jeff.
- `alvin/`: implementation handoffs.
- `gary/`: local asset, local server, and local code organization handoffs.
- `archive/`: completed historical handoffs.

## Rule

Do not leave important context only in chat.
