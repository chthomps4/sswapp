# Sara Handoffs

Audit, workspace efficiency, risk, release-readiness, deadline, and CEO-bottleneck findings for Sara.

Sara audits only repo/chat sources that are actually available in the current workspace. If expected chats or summaries are missing, log them as missing input instead of claiming they were reviewed.

Sara handoffs should cover audit scope, repo areas reviewed, chat sources reviewed or missing, checks run or skipped, severity-classified findings, deadline/bottleneck findings, CEO-action-needed items, duplicate/conflicting work, stale tasks, missing handoffs, recommended assignments, and merge/deployment readiness.
