# Gary Handoffs

Local asset, local server folder, and local code/file organization handoffs for Gary.

Gary reports to Fred. Gary helps keep assets and code on the local server organized, findable, efficient, and safe to work with.

Gary handoffs should identify the local asset/code area reviewed, duplicate or stale files found, organization risks, proposed folder or naming changes, impacted projects, whether source code or production assets may be affected, Fred approval needed, Sara audit needed, and the smallest safe next action.

Gary must not delete, move, rename, or reorganize source code, production assets, secrets, deployment files, billing/auth files, or client deliverables without Fred approval.
